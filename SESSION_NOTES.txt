2026-06-16 - Initial project review
-----------------------------------
Goal:
- Understand the web frontend project only, from a FE developer perspective.

Findings:
- This is a Vite + React + TypeScript SPA using TanStack Router and TanStack Query.
- Dev server runs on port 5173.
- Frontend API calls go through /api/*.
- Vite dev proxy sends /api requests to API_PROXY_TARGET, defaulting to http://localhost:8080.
- Login is not hardcoded in the real app. The login form posts tenant_slug, email, and password to POST /api/auth/login, then fetches GET /api/auth/me.
- Hardcoded login responses exist only in Playwright e2e mocks under e2e/mock-api.ts.
- Some README/comments/UI strings show mojibake encoding artifacts, but typecheck still passes.

Verification:
- npm.cmd run typecheck passed.
